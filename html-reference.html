    Description

    A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading

    Syntax

    You can use H tag when you are writing the code. The numbers 1-6 can be followed after the H tag. These numbers tell you how big the heading is going to be. Look at the display and the example to help you understand the context better.

    Title

    Description

    The title tag is required in all HTML documents and it defines the title of the document.

    Syntax

    The title tag can be used in the code to have titles. You must always have a start title tag and an end title tag. The title tag can ONLY BE USED ONCE in the entire code.

    Paragraph

    Description

    Paragraphs are usually represented in visual media as blocks of text that are separated from adjacent blocks by vertical blank space and/or first-line indentation

    Syntax

    P tags can be used in the code to make up paragraphs. You always have a start p tag and an end p tag
