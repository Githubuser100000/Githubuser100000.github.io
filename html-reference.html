Heading

A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading

Syntax

    Start Tag: <H1>

    End Tag: </H1>

    You can use numbers 1 through 6 to change the size of the heading like this: <H2>Hi</H2> so it becomes like this

    Hi

Example

    <H1>This is a heading </H1>

    <H2>This is the second heading</H2>

Display

This is a heading

This is the second heading

Title

Description:

The title tag is required in all HTML documents and it defines the title of the document.The title tag can be used in the code to have titles. You must always have a start title tag and an end title tag. The title tag can ONLY BE USED ONCE in the entire code.

Syntax

    Start Tag: <title>

    End Tag: </title>

    Example

    <title>This is the title of the document</title>

Element:Tables

Description

An HTML table is defined with the table tag.Each table row is defined with the table row tag. A table header is defined with the table header tag. By default, table headings are bold and centered. A table data/cell is defined with the table data tag.

Syntax

    Start tag: <table>

    Table Header tag: <th> and </th>

    Table Row tag: <tr> and </tr>

    Table Data tag: <td> and </td>

    End tag: </table>

Example

<table>
<tr>
<th>Owner</th>
<th>Name</th>
<th>Cat/Dog</th>
</tr>
<tr>
<th>Sanjana</th>
<th>Maggie</th>
<th>Dog</th>
</tr>
<tr>
<th>Sanjana</th>
<th>Genledyn</th>
<th>Dog</th>
</tr>
</table>

Display

This is how the code looks on the website:

Owner Name Cat/Dog
Sanjana Maggie Dog
Mia Geneldyn Cat
